# H&M

> 리액트 & json-server를 공부하면서 만들어본 H&M !!

## Json-server

- 설치 <br/>
  `npm i json-server`

- 시작 <br/>
  - root 경로에 `db.json` 파일을 만들어준다.
  - db.json 파일안에 json형태로 데이터를 만들어준다.
  - `json-server --watch db.json --port 5000` 를 통해 서버를 실행한다. <br/>
  - 주의!! json-server 포트와 개발서버(localhost) 포트와 같으면 안된다.
